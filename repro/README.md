# pi.cmd Windows wrapper: multi-line prompts corrupt command line

## Root Cause

`pi.cmd` (the Windows batch wrapper generated by npm) passes the `-p` prompt through
`cmd.exe`, which corrupts prompts containing blank lines (`\n\n`). The prompt is
truncated at the first blank line, and subsequent arguments (`--session`, etc.) may
also be affected.

**This is a Windows cmd.exe limitation, not a pi bug.**

## Reproduction

```bash
mkdir -p repro/src
echo 'version = "1.0.0"' > repro/src/single.py

# prompt-bug.txt: blank line between paragraphs
cat > repro/prompt-bug.txt << 'EOF'
Edit src/single.py. Change the version from 1.0.0 to 2.0.0-beta1.

The file should still have exactly one line.
EOF
```

### Fails: using pi.cmd wrapper (cmd.exe)

```bash
pi.cmd -p "$(cat repro/prompt-bug.txt)" \
       --model deepseek-v4-flash \
       --session repro/session-picmd.jsonl

# Session file NOT created (cmd.exe corrupted the args)
ls -la repro/session-picmd.jsonl  # file not found
```

### Works: using node directly (bypasses cmd.exe)

```bash
PI_CLI="$(dirname $(dirname $(which pi.cmd || echo /nope)))/node_modules/@earendil-works/pi-coding-agent/dist/cli.js"

node "$PI_CLI" -p "$(cat repro/prompt-bug.txt)" \
               --model deepseek-v4-flash \
               --session repro/session-node.jsonl

ls -la repro/session-node.jsonl  # file exists, ~5KB
```

### Also works: single-line prompts with pi.cmd

```bash
echo 'Edit src/single.py. Change the version from 1.0.0 to 2.0.0-beta1.' > repro/prompt-ok.txt

pi.cmd -p "$(cat repro/prompt-ok.txt)" \
       --model deepseek-v4-flash \
       --session repro/session-ok.jsonl

ls -la repro/session-ok.jsonl  # file exists — single-line prompts are fine
```

## Fix Applied

The benchmark runner now locates `cli.js` from pi.cmd's install directory and
invokes it via `node` directly, bypassing the Windows batch wrapper:

```python
pi_cmd = shutil.which("pi.cmd")
pi_dir = Path(pi_cmd).parent
cli_js = pi_dir / "node_modules" / "@earendil-works" / "pi-coding-agent" / "dist" / "cli.js"
cmd = ["node", str(cli_js), "-p", prompt, ...]
```
